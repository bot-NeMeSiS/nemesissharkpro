# V188 Advanced Stats PRO

Incluye:
- ROI, winrate, profit, stake total y cuota media si existen datos.
- Cobertura de marcador/cuotas en fixtures.
- Ligas y mercados más usados.
- Readiness para Machine Learning real.
- Snapshots de estadísticas.

Rutas:
- `/advanced-stats-pro`
- `/cliente/advanced-stats`
- `/admin/advanced-stats`

APIs:
- `/api/v188/advanced-stats`
- `/api/v188/advanced-stats/snapshot`
- `/api/v188/ml-readiness`

Política:
- No inventa xG, posesión, tiros ni eventos si la API no los trae.
- Calcula métricas sobre SQLite real/caché existente.
