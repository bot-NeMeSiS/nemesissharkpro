## Forma recomendada de subir

No uses arrastrar y soltar de GitHub web si te falla. Usa GitHub Desktop o terminal:

```bash
git add .
git commit -m "V81 ultra clean"
git push
```

Si ya tienes repo:
1. Borra todo el contenido del repo local excepto `.git`.
2. Copia dentro el contenido de este ZIP.
3. Ejecuta los comandos de arriba.

No mezcles esta build con carpetas viejas ni otros ZIPs.
