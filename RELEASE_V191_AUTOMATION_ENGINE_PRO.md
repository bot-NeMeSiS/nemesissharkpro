# RELEASE V191 · AUTOMATION ENGINE PRO

Este avance convierte la capa de datos V190 en un sistema automatizable.

## Qué mejora

- Menos trabajo manual en admin.
- Más histórico guardado para ML futuro.
- Mejor velocidad percibida por cache warming.
- Más control operativo desde un panel premium.
- Telegram preparado para resúmenes automáticos reales.
- Cron listo para Render o llamadas externas seguras.

## Importante

La automatización no inventa datos. Si una API, token, chat ID o base URL no está configurada, el job queda en `warning` con explicación en el panel.
